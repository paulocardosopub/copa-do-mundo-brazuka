# Copa do Mundo Brazuka

Jogo web/mobile de gerenciamento idle de futebol 3D com temática brasileira fictícia.

## MVP jogável

- Criação inicial do jogador
- Home com boneco 3D, recursos, OVR Brazuka e próximo jogo
- Treinos por atributo com custo, energia, XP e evolução
- Cálculo de OVR ponderado por posição
- Partidas automáticas com placar, narração, eventos e habilidades manuais
- Resultado com recompensas de XP, Reais Brazuka, fama e torcida
- Carreira com fases, Copa Brazuka inicial, rivais e desbloqueios
- Equipamentos, patrocinadores, missões, feed fictício e configurações
- Save local com progresso offline limitado
- Workflow de GitHub Pages em `.github/workflows/deploy.yml`

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
